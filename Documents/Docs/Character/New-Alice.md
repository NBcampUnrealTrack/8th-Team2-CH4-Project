# 이상한 나라의 앨리스 (Alice)

역할군 : 원거리 딜러

스탯 : HP 180

배경 및 기획 의도 (동화의 연대):
조급한 회중시계 토끼, 하트 여왕의 트럼프 카드 병정, 자라나는 장미 덩굴, 미친 모자장수의 티타임 등 여러 원작 요소를 이용해 전장의 공간과 인지 구조를 뒤흔드는 세련된 원거리 딜러입니다.
기본 공격 피해가 강한 원거리 딜러에, 다양한 상태이상과 생존기를 가지고 있으면서 피해량도 높은 대신
체력이 180으로 평균보다 낮고, 카드가 투사체라 빗나갈 위험이 있어 이를 상태이상 스킬로 만회해야 합니다.

# [스킬 셋 메커니즘]
[LMB] 평타 - 로얄 카드 :
앨리스가 오른손에 쥔 하트 여왕의 예리한 트럼프 카드들을 던집니다.
카드는 벽이나 지형에 부딪히면 1회 튕기며, 숨어있는 적에게도 온전히 피해를 가할 수 있습니다.
피해량 30 / 공격속도 1.0

## [RMB] 보조 공격 - 시계 토끼 :
앨리스가 전방으로 "늦었어, 늦었다고!" 라고 외치며 달리는 회중시계 토끼의 환영을 직선으로 빠르게 내보냅니다. 토끼 환영은 경로상의 적들을 관통하며 30의 피해를 주고, 적중된 적의 발밑에 시계 태엽 인장을 새겨 2초동안 둔화시킵니다.
피해량 30 / 재사용 대기시간 10초

## [Shift] 기술 1 - 토끼의 약 :
2초동안 몸이 작아져 히트박스가 감소하고, 이동 속도 50% 증가 효과를 받습니다. 시간이 끝나면 원래 모습으로 돌아옵니다.
피해량 0 / 재사용 대기시간 15초

## [E] 기술 2 - 하트 여왕의 사형 선고 :
5초동안 기본 공격의 피해량이 30% 감소하지만, 공격 속도가 2배가 됩니다.
피해량 0  / 재사용 대기시간 15초
기본 공격 피해량 21 / 공격속도 2.0으로 변경

## [Q] 궁극기 - 매드 티타임 (The Mad Tea Party) :
"아직 6시야! 무대는 끝나지 않아!"
앨리스가 시계를 깨뜨리며 자신을 중심으로 시간 정지를 전개합니다. 영역 내의 모든 적은 시간이 멈춰 기절 상태에 빠집니다. 영역은 2초간 유지되며 앨리스 본인은 1초 뒤부터 움직일 수 있습니다.

# 예상 TTK
기본 공격만으로도 5초동안 150의 피해를 누적할 수 있으며,
하트 여왕의 사형 선고를 사용할 경우 210,
여기에 평타를 캔슬하며 시계토끼를 사용할 경우 240의 피해를 누적할 수 있습니다.
증강을 모두 공격적으로 골랐을 땐, 원거리 딜러지만 위험을 감수하고 E를 쓰고 쉬프트로 파고들어서
궁을 맞추고 3초동안 168의 피해를 누적해 TTK가 3초까지 짧아질 수 있습니다.

# 증강 선택
생존성을 강화하는 시계 토끼 증강과, 피해량을 강화하는 하트 여왕 증강으로 나뉩니다.

## 1차 증강
- 증강 : 시간이 약
대상 스킬 : [Shift] 기술 1 - 토끼의 약
증강 효과 : 시간을 가속해 이동속도 증가 효과가 30% 추가로 증가합니다.

- 증강 : 재판의 시간
대상 스킬 : [Q] 궁극기 - 매드 티타임 (The Mad Tea Party):
증강 효과 : 궁극기 시전 시, 즉시 움직일 수 있습니다.

## 2차 증강
- 증강 : 지각 직전
대상 스킬 : [RMB] 보조 공격 - 시계 토끼
증강 효과 : 시계 토끼의 투사체 속도가 빨라지며, 시계 토끼가 더 이상 적을 관통하지 못합니다.
대신 시계 토끼가 적과 충돌하거나 끝까지 이동할 경우 폭발해 30의 피해를 추가로 가하고, 1초동안 기절시킵니다.

- 증강 : 심기 불편
대상 스킬 : [E] 기술 2 - 하트 여왕의 사형 선고
증강 효과 : 사용 시, 기본 공격이 약간 벌어진 채 두 발씩 나갑니다. 각각의 공격은 여전히 1회 튕깁니다.
