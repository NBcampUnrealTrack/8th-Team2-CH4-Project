# Fairy Tale Selection Criteria

## 기준

FieryTale에 채용할 동화 후보는 다음 기준으로 선별한다.

- 전 세계적으로 인지도가 높거나, 특정 문화권을 대표할 정도로 강한 상징성이 있을 것
- 원전, 고전 번역, 민담 채록본 등 오래된 자료를 기반으로 재해석할 수 있을 것
- 현대 애니메이션, 영화, 게임, 뮤지컬의 고유 디자인과 설정을 피할 수 있을 것
- 캐릭터, 구조물, 몬스터, 스킬, 맵 테마로 확장하기 쉬울 것
- AOS/TPS 전투 캐릭터로 변환했을 때 역할과 능력 콘셉트가 명확할 것

## 저작권/상표권 주의

이 문서는 법률 자문이 아니라 기획용 리서치 문서다. 실제 상업 출시 전에는 대상 국가 기준으로 별도 검토가 필요하다.

일반적으로 오래된 민담, 그림 형제, 샤를 페로, 한스 크리스티안 안데르센, 이솝, 천일야화, 루이스 캐럴, 카를로 콜로디 등의 원전은 퍼블릭 도메인으로 활용 가능한 경우가 많다. 다만 특정 현대 각색판의 캐릭터 디자인, 조연, 노래, 대사, 장면 구성, 로고, 제목 처리, 상품명은 별도 권리의 대상이 될 수 있다.

특히 디즈니 등 현대 스튜디오의 버전은 다음 요소를 피해야 한다.

- 고유 의상 색 조합과 실루엣
- 현대 각색에서 추가된 조연 캐릭터
- 영화판의 성격, 말투, 개그, 상징 소품
- 곡명, 가사, 시각적 장면 구성
- 상표로 기능하는 로고, 제목 처리, 캐릭터 상품 이미지

민담과 전통문화 표현은 저작권과 별도로 문화적 맥락, 공동체 존중, 현대 전승자 권리 문제가 있을 수 있다. 특정 원주민/현대 공동체의 살아 있는 전승은 사용 전 추가 검토가 필요하다.
