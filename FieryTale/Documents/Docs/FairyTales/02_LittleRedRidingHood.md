# Little Red Riding Hood

## 원전/권리 메모

샤를 페로와 그림 형제 계열의 오래된 민담을 기반으로 한다. 현대 삽화의 특정 빨간 망토 디자인이나 영화식 늑대인간 설정은 피한다.

## 짧은 스토리

소녀가 할머니에게 음식을 전하러 숲을 지나가다 늑대에게 속는다. 늑대는 할머니를 먼저 공격한 뒤 소녀까지 속이려 한다. 버전에 따라 사냥꾼이 구출하거나, 더 어두운 결말로 끝난다.

## 주요 특징

- 숲길, 빨간 망토, 늑대, 바구니, 할머니 집
- 순진함과 포식자의 기만이라는 명확한 대비
- 추적, 위장, 함정, 시야 플레이와 잘 맞음

## 캐릭터 후보

- Red Riding Hood: 기동형 원거리 딜러, 덫과 추적 표식 사용
- Wolf: 은신, 변장, 급습에 특화된 암살자
- Hunter: 구조, 견제, 장거리 사격형 캐릭터

## 게임 적용 아이디어

숲 라인, 사냥꾼 오두막, 늑대의 은신로를 맵 테마로 만들기 좋다. Red와 Wolf를 라이벌 캐릭터로 설계하면 초반 캐릭터 간 서사가 강해진다.
